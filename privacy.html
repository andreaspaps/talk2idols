Functional Features and Paid/Free Functionality:

  • In the "Google speech key" field insert your Google Cloud text-to-speech API key. To get your Google text-to-speech API key, follow these steps:

    Go to Google Cloud and sign up for the free tier with a Google account, you may need to add a credit/debit card to activate the account. Google's text-to-speech offers 1 Million characters (200,000 words) free per month which is plenty for chatbot use and another 1M after for 3$.

    - Log in to your GCP account and navigate to the GCP console.
    - Click on the project dropdown and create a new project or select an existing one.
    - In the left sidebar, click on APIs & Services > Library.
    - Search for Text-to-Speech API and click on the result.
    - Click Enable to enable the API for your project.
    - In the main console left sidebar, click on Credentials and you will see the API key or click "Create credentials" and then select API key. Tap the "Show" and copy the key.

  • Click the "Clear API key" to remove the stored Keys from local storage in case you want to. Uninstalling the app automatically removes all stored keys. We do NOT collect any form of data as it is stored locally on the device.

How to use:

  • Tap "settings" (top right gear icon). During the initial setup, you only need to input the API keys and your name once, and they will be saved in the local storage.

  • After obtaining the Key from OpenAI's website, copy-paste the key in the settings menu and you are ready to chat with the chatbot (type your question/anser in the "ask anything" input field).

  • To chat type into the "ask anything" input field and tap "submit" or tap the "speak" button to start the speech recognition. You must manually enable the app's "microphone permission" in your phone's "app permissions" setting.

  • Google's text-to-speech Key enables the chatbot to speak in both male and female voice.

  • You can chat to the bot directly or you can insert the name of any famous person in the settings menu. The chatbot will reply as if it was that person.

  • Tap the "refresh" button (top left icon) every time you choose a different personality in order to reset the memory.

  • Starting with "/" in the "ask anything" input field mutes the chatbot's reply (only when text-to-voice is enabled).

  • Starting with "Search" performs an internet search. You need a free Custom Google Search API key. First create a free programmable search engine here, then click "Add" enter a name and click "Create". Then click at the "Get a key" blue button here. Select "Create project" and then click "Show". Copy the API key and paste it in the settings menu "Google search key".
